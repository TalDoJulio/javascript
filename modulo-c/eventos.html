Eventos é tudo que pode acontecer com o elemento ( nesse caso essa DIV ), nessa aula falaremos sobre eventos de mouse.

mouseente = Vai disparar um evento quando o mouse entra dentro do elemento.

mousemove = Mover o mouse dentro do elemento.

mousedown = Esse evento ocorre quando deixa precionado o mouse dentro do elemento.

mouseup = Dispara quando o solta o botao do mouse

click = Quando clica com o mouse no elemento.

mouseout = Quando eu arrasto o mouse para fora do elemento.

Existem outros eventos que podem ser usados, basta acessar o site da mozilla

Uma função em JavaScript, é um conjunto de codigos que vao ser executadas so quando o evento acontecer
